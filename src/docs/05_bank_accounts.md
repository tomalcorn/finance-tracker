# Bank Accounts

Bank accounts show the live balance for each account in the app.

## Metrics Tab

The metrics tab gives a quick summary card for each account. It is the fastest
way to answer:

- what accounts do I have?
- what is each balance right now?

## Table Tab

The table tab is the editable view. It lets you:

- add new accounts
- edit the starting balance
- rename an account

## How To Reconcile An Account

1. Start with the opening balance.
2. Enter all known payments.
3. Check the `Checked` status on payments once you have verified them.
4. Compare the computed `Current Balance` with the real statement balance.

If the numbers do not match, the usual causes are:

- a missing payment
- a payment linked to the wrong bank account
- a payment date that falls outside the date you expected

## Tips

- Keep account names short and obvious.
- Use the metrics tab when you want a quick glance.
- Use the table tab when you need to fix the opening balance or tidy the setup.
